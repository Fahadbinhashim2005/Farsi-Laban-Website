🍰 Farsi Laban

Premium Dessert Brand Website

A modern and visually appealing dessert brand website designed to deliver a premium digital experience. Built with a focus on elegant UI, responsive design, smooth interactions, and modern frontend development practices.

🌐 Live Demo: https://farsi-laban-website.vercel.app

---

✨ Overview

Farsi Laban is a premium dessert brand website that showcases signature desserts through a clean, modern, and engaging user interface. The project focuses on creating a luxury-inspired browsing experience while maintaining excellent performance and responsiveness across devices.

The website combines modern design trends such as glassmorphism, smooth animations, and mobile-first responsiveness to create an immersive user experience.
